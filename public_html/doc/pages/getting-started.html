Minimal JS, Maximal Functionality

Why repeat CRUD functionalities for different applications all the time?!
Wouldn't it be just great to have a framework that does all the CRUD operations and therefore leaves us to focus on the less mundane things?

Introducing ThisJS

With ThisJS, CRUD is its core of operations. And it's easy to set up in 3 short steps:

STEP 1:
Add Framework

<script src="/path/to/this.js" />

No dependencies!

STEP 2:
Initialize App

var app = new ThisApp('container_selector')

STEP 3:
Specify Base URL

app.setBaseURL('http://path/to/all/resources/')

STEP 4:
Start the app

app.start()

... and you're done!
